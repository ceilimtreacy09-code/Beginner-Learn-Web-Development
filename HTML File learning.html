HTML — The Skeleton

HyperText Markup Language · Lesson 1 of 4

What is HTML?

HTML is the structure of every webpage you've ever visited. Think of it like the frame of a building — it defines what exists on the page and where it goes, but doesn't make it look pretty (that's CSS's job).

HTML uses elements — blocks of code wrapped in angle bracket tags like <p> for paragraph or <h1> for a heading.

Key idea: Every HTML element has an opening tag <tag> and a closing tag </tag>. The content goes in between.

The anatomy of an HTML page

HTML · index.html 
<!-- Every HTML file starts with this declaration -->
<!DOCTYPE html>

<html lang="en">

  <head>
    <!-- Meta info — not visible to the user -->
    <meta charset="UTF-8">
    <title>My First Website</title>
  </head>

  <body>
    <!-- Everything visible goes inside <body> -->
    <h1>Hello, World!</h1>
    <p>This is my first paragraph.</p>
  </body>

</html>

TagWhat it doesExample

Attributes — giving elements information

Attributes are extra data you add inside the opening tag. They always follow the pattern: attribute="value"

HTML · Attributes 
<!-- href tells the link WHERE to go -->
<a href="https://google.com" target="_blank">Open Google</a>

<!-- src is the image file, alt is the description -->
<img src="cat.jpg" alt="A cute cat" width="300">

<!-- class is used to apply CSS styling later -->
<p class="intro-text">This paragraph has a class!</p>

CSS — The Style

Cascading Style Sheets · Lesson 2 of 4

What is CSS?

CSS is how you make websites look good. It controls colours, fonts, spacing, layout, animations — everything visual. Without CSS, every website looks like a plain text document.

CSS works by selecting HTML elements and applying rules (property: value pairs) to them.

The syntax: selector { property: value; } — Select what, then declare how it should look.

The 3 ways to add CSS

1
Inline — directly on the element (avoid this, hard to manage)

<p style="color: red;">Text</p>

2
Internal — inside a <style> tag in your HTML file

Good for single-page projects.

3
External — a separate style.css file linked with <link>

✅ Best practice. Keeps code organised.

CSS · style.css — Selectors & Properties 
/* ---- Selectors ---- */

/* 1. Element selector — targets ALL <p> tags */
p {
  color: #333;
  font-size: 16px;
}

/* 2. Class selector — targets elements with class="btn" */
.btn {
  background: purple;
  color: white;
  padding: 10px 20px;
  border-radius: 8px;
}

/* 3. ID selector — targets the element with id="hero" */
#hero {
  font-size: 48px;
  font-weight: bold;
}

The Box Model — critical concept

Every HTML element is a box. CSS controls 4 layers around that box:

MARGIN (outer space)
BORDER
PADDING (inner space)
CONTENT
CSS · Box Model in practice 
.card {
  /* Content size */
  width: 300px;
  height: 150px;

  /* Padding: space INSIDE the border */
  padding: 20px;

  /* Border: the line around the element */
  border: 2px solid #7c3aed;
  border-radius: 12px;

  /* Margin: space OUTSIDE the element */
  margin: 16px;

  /* Background colour */
  background: #f5f5ff;
}

Flexbox — modern layout

Flexbox is how you arrange elements in a row or column. It's the most useful CSS tool to learn first. Just apply display: flex to a parent container.

CSS · Flexbox 
.container {
  display: flex;

  /* Direction: row (left→right) or column (top→bottom) */
  flex-direction: row;

  /* Space between items */
  gap: 16px;

  /* Align along the MAIN axis (horizontal when row) */
  justify-content: center;   /* or: flex-start, flex-end, space-between */

  /* Align along the CROSS axis (vertical when row) */
  align-items: center;
}

.container div {
  flex: 1; /* Each child shares the space equally */
}

JavaScript — The Logic

Dynamic Scripting · Lesson 3 of 4

What is JavaScript?

JavaScript (JS) makes websites interactive. Without JS, a webpage is just a static document. With JS, you can respond to button clicks, show/hide content, validate forms, fetch data from the internet, create games — anything.

HTML is the structure, CSS is the style, JavaScript is the behaviour.

Variables — storing data

JavaScript · Variables 
// let = can be changed later
let username = "Sarah";
let age = 16;
let isStudent = true;

// const = cannot be changed (constant)
const PI = 3.14159;
const siteName = "My Website";

// Printing to the console (press F12 to see it)
console.log(username); // → "Sarah"
console.log(age + 4);   // → 20

If/Else — making decisions

JavaScript · Conditionals 
let score = 72;

if (score >= 90) {
  console.log("A grade — excellent!");
} else if (score >= 70) {
  console.log("B grade — good work");
} else {
  console.log("Keep studying!");
}
// → "B grade — good work"

Functions — reusable blocks of code

JavaScript · Functions 
// Define a function
function greetUser(name) {
  return "Welcome, " + name + "!";
}

// Call the function
let message = greetUser("Sarah");
console.log(message); // → "Welcome, Sarah!"

// Modern arrow function (same thing, shorter syntax)
const double = (n) => n * 2;
console.log(double(5)); // → 10

The DOM — making JS talk to HTML

The DOM (Document Object Model) is how JavaScript reads and changes your HTML. You can grab any element by its ID or class, then change its text, style, or content.

JavaScript · DOM Manipulation 
// HTML: <h1 id="title">Hello</h1>
//       <button id="myBtn">Click me</button>

// Grab the element by its ID
const title = document.getElementById("title");
const btn = document.getElementById("myBtn");

// Change its text content
title.textContent = "Text changed by JavaScript!";

// Change its style
title.style.color = "purple";

// Listen for a click on the button
btn.addEventListener("click", function() {
  title.textContent = "You clicked the button!";
  title.style.fontSize = "2rem";
});

Build Your First Project

Putting it all together · Lesson 4 of 4

🎯 PROJECT CHALLENGE

Build a Personal Intro Card

Combine everything you've learned — HTML structure, CSS styling, and JavaScript interactivity — into one complete mini-project.

  • A name heading + short bio paragraph
  • A styled card with colours, padding, border-radius
  • A button that shows/hides extra info using JavaScript
HTML + CSS + JS · Complete project file 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Intro Card</title>
  <link rel="stylesheet" href="style.css">
</head>

<body>
  <div class="card">
    <div class="avatar">👩‍💻</div>
    <h1>Sarah Connelly</h1>
    <p>Aspiring web developer from Dublin 🇮🇪</p>
    <div class="extra-info" id="moreInfo">
      📚 Learning HTML, CSS & JavaScript
    </div>
    <button id="toggleBtn">Show More About Me</button>
  </div>
  <script src="script.js"></script>
</body>
</html>
